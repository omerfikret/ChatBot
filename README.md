Kullanıcıdan veri girişi aldığında önce Linear SCV yöntemi ile intent ayıran sonra da fuzzywuzzy kütüphanesi ile benzerlik oranından cevap bulan bir Osmanlı Tarihi botu yaptım. Öncelikle fuzzywuzzy yerine kosinüs benzerliği
kullanacaktım ama cümlelerin kısalığı bu yöntemi kullanmamı zorlaştırdı. Genel olarak ml botlarının işleyişi, vektörleştirme, pandas, dataset düzenleme konusunda çok şey öğrendim. Bu botu birçok soru cevap örneğiyle eğitip 
çeşitlendirebilirim. Şuanlık birkaç hatayla bırakıp sonraki projeye botun daha da gelişmiş versiyonlarıyla devam edeceğim.
